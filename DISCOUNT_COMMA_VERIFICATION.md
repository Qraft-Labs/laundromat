    ```
    - Discount percentage saved
    - Discount amount saved
    ```

    ## 8. Issues Found & Fixed

    ### Issue: 170 Orders Had Discount Not Applied

    **Problem:**
    - Orders had discount_amount stored
    - But total_amount wasn't reduced
    - Formula broken: total = subtotal (should be subtotal - discount)

    **Example:**
    ```
    Order ORD20260571 (BEFORE FIX):
    Subtotal:     UGX 1,242,000
    Discount:     UGX 186,300
    Total (WRONG): UGX 1,242,000 ❌
    Should be:    UGX 1,055,700

    Order ORD20260571 (AFTER FIX):
    Subtotal:     UGX 1,242,000
    Discount:     UGX 186,300
    Total:        UGX 1,055,700 ✅
    ```

    **Fix Applied:**
    ```sql
    UPDATE orders 
    SET 
    total_amount = subtotal - discount_amount,
    balance = (subtotal - discount_amount) - amount_paid,
    payment_status = CASE
        WHEN (subtotal - discount_amount) - amount_paid <= 0 THEN 'PAID'
        WHEN amount_paid = 0 THEN 'UNPAID'
        ELSE 'PARTIAL'
    END
    WHERE discount_amount > 0 
    AND total_amount = subtotal;
    ```

    **Result:**
    - ✅ Fixed 170 orders
    - ✅ All discounts now reflected in totals
    - ✅ All balances recalculated
    - ✅ All payment statuses updated
    - ✅ 100% accuracy verified

    **Backend (order.controller.ts):**
    ```typescript
    // Validate discount
    if (discount_percentage < 0 || discount_percentage > 50) {
    throw new Error('Discount must be 0-50%');
    }

    // Fetch actual prices from database
    const priceResult = await query(
    'SELECT price FROM price_items WHERE id = $1',
    [item.price_item_id]
    );
    const actualPrice = priceResult.rows[0].price;

    // Calculate everything
    const subtotal = items.reduce((sum, item) => 
    sum + (item.quantity * actualPrice), 0
    );
    const discount_amount = Math.round(subtotal * (discount_percentage / 100));
    const total_amount = subtotal - discount_amount;
    const balance = total_amount - amount_paid;

    // Store in database
    await query(
    `INSERT INTO orders (subtotal, discount_percentage, discount_amount, 
    total_amount, balance, ...) VALUES ($1, $2, $3, $4, $5, ...)`,
    [subtotal, discount_percentage, discount_amount, total_amount, balance, ...]
    );
    ```
